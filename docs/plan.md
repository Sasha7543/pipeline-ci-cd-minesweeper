# План виконання лабораторної роботи

## Обрана гра

Для стандартного рівня обрано гру **Minesweeper** з графічним інтерфейсом на Pygame.

## Функціональні вимоги

- Користувач бачить ігрове поле з клітинками.
- Користувач відкриває клітинку лівою кнопкою миші.
- Користувач ставить або прибирає прапорець правою кнопкою миші.
- Якщо відкрита міна, гра переходить у стан поразки.
- Якщо відкриті всі безпечні клітинки, гра переходить у стан перемоги.
- Перший клік не повинен відкривати міну.
- Гру можна перезапустити без повторного запуску програми.
- Через аргументи командного рядка можна змінити складність, розмір поля, кількість мін, розмір клітинки та колір фону.

## Нефункціональні вимоги

- Код має бути зрозумілим і розділеним на модулі.
- Логіка гри має тестуватися окремо від графічного інтерфейсу.
- Проєкт має запускатися після встановлення залежностей з `requirements.txt`.
- Репозиторій має містити документацію для етапів Plan, Code, Build.
